# Open‑Meteo Weather Cards
Avancerede vejrkort og REST‑sensorer til Home Assistant baseret på Open‑Meteo API, egen vejr station og Blitzortung lyn integration
Projektet giver hyperlokale vejrdata, lyn‑detektion, skybrudsrisiko, sigtbarhed, solskin, barometerstatus og en fuld 5‑dages prognose — helt uden API‑nøgler.

---

## ⭐ Funktioner

- REST‑sensorer baseret på Open‑Meteo (current, hourly, daily)
- Vejret lige nu (ikon, temperatur, vind, nedbør)
- 5‑dages vejrudsigt
- Atmosfærisk stabilitet (CAPE, LI)
- Barometer status og tendens
- Forventet nedbør (15 / 30 / 60 min)
- Lyn detektion
- Max & Min temperatur for dagen
- Sigtbarhed og trafik
- Skybruds risiko
- Solskin i dag (soltimer)

# Open‑Meteo Weather Cards

## 🧩 Kortoversigt

### Vejret lige nu
Live‑data fra Open‑Meteo med temperatur, vind, nedbør og ikon.

### 5‑dages vejrudsigt
Max/min temperatur, nedbør, vind og ikon for de næste fem dage.

### Atmosfærisk stabilitet
CAPE, Lifted Index og stormindikator.

### Barometer status
Tryk + tendens (stigende, faldende, stabil).

### Forventet nedbør (15 / 30 / 60 min)
Nowcast‑baseret intensitet med farvekoder.

### Lyn detektion
Afstand, retning og antal registrerede lyn.

### Max & Min temperatur for dagen
Udregnet fra egen vejr station, og 4 hjælpere

### Sigtbarhed og trafik
Sigtbarhed i meter + trafikrelevans (tåge, dis, regn).

### Skybruds risiko
Konvektiv nedbør + intensitet.

### Solskin i dag
Soltimer.
